# Hull Connector Generator

We use [Builder](http://formidable.com/open-source/builder/)

## Installing

```
~/test ❯❯❯ yarn global add builder-init hull-connector hull-connector-dev;
~/test ❯❯❯ builder-init hull-connector;
```
